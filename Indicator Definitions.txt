Below are the **configs (parameters) for each indicator module** in your current setup, what they do, and what columns you get in the **output feature DF** (`df_feat`). I’ll use the convention:

* **Tagging:** every column is prefixed as `"{tag}__{COL}"` so you can run the same indicator multiple times with different configs.
* **Where it plots:** **Overlay** = on price chart, **Panel** = below chart.

---

## RSI with Divergence (`name="rsi_divergence"`) — Panel + optional price markers

### Configs

* `length` (int, default `14`)
  RSI period.

* `pivot_lookback` (int, default `5`)
  Pivot detection window. Higher = fewer pivots/divergences (more conservative).

* `min_rsi_delta` (float, default `2.0`)
  Minimum RSI difference between the two pivot points required to count as divergence.

* `os_level` (float, default `20`)
  Oversold threshold (line + zone lower bound).

* `ob_level` (float, default `80`)
  Overbought threshold (line + zone upper bound).

* `zone_mode` (str, default `"cross"`)
  Controls which divergences are **considered valid** relative to `os_level/ob_level`:

  * `"cross"`: divergence segment must cross the threshold level
  * `"touch"`: at least one endpoint is beyond the threshold
  * `"both"`: both endpoints beyond threshold (strongest)
  * `"none"`: no zone filter (accept all)

* `show_zone_shading` (bool, default `True`)
  Shades OB/OS zones in the RSI panel.

* `show_levels` (bool, default `True`)
  Draws emphasized horizontal lines at `os_level` and `ob_level`.

* `show_rsi_div_lines` (bool, default `True`)
  Draw divergence lines **inside RSI panel**.

* `mark_price` (bool, default `True`)
  Shows divergence markers on the main price chart (offset from candles).

* `price_marker_size` (int, default `16`)
  Size of divergence markers on main chart.

* `marker_offset_window` (int, default `14`)
  Used to compute average candle range for offset.

* `marker_offset_mult` (float, default `0.9`)
  Offset magnitude (higher = markers farther away from candles).

### Output DF columns (per tag)

* `{tag}__RSI`
* `{tag}__BULL_DIV` (bool, true where bull divergence confirmed)
* `{tag}__BEAR_DIV` (bool)
* `{tag}__BULL_START_RSI` (RSI at first pivot for bull div)
* `{tag}__BEAR_START_RSI` (RSI at first pivot for bear div)

## Stochastic (`name="stochastic"`) — Panel

### Configs

* `k_length` (int, default `14`)
  Lookback for highest high / lowest low.

* `d_length` (int, default `3`)
  Smoothing for %D (moving average of %K).

* `smooth` (int, default `3`)
  Smoothing applied to raw %K before %D.

* `show_levels` (bool, default `True`)
  Draws dotted 20/80 lines on the stochastic panel.

### Output DF columns

* `{tag}__K`
* `{tag}__D`

## Bollinger Bands (`name="bollinger_bands"`) — Overlay on price

### Configs

* `length` (int, default `20`)
  Middle band = SMA(length).

* `stdev` (float, default `2.0`)
  Upper/lower = mid ± stdev * rolling_std.

* `fill` (bool, default `True`)
  Fills the area between upper/lower bands.

### Output DF columns

* `{tag}__MID`
* `{tag}__UP`
* `{tag}__LO`

---

## Momentum (`name="momentum"`) — Panel

### Configs

* `length` (int, default `10`)
* `mode` (str, default `"diff"`)

  * `"diff"`: close - close.shift(length)
  * `"roc"`: % return: 100 * (close / close.shift(length) - 1)

### Output DF columns

* `{tag}__MOM`

---

## Supertrend (`name="supertrend"`) — Overlay on price + buy/sell signals + markers

### Configs

* `length` (int, default `10`)
  ATR length.

* `multiplier` (float, default `3.0`)
  ATR multiplier.

* `show_markers` (bool, default `True`)
  Show BUY/SELL markers on price chart.

* `marker_size` (int, default `18`)
  Marker size.

* `marker_offset_window` (int, default `14`)
  Window for average candle range (offset calculation).

* `marker_offset_mult` (float, default `1.2`)
  Offset magnitude for markers (higher = further away).

### Output DF columns

* `{tag}__ST` (supertrend value)
* `{tag}__DIR` (1 for uptrend, -1 for downtrend)
* `{tag}__BUY` (bool on uptrend flip)
* `{tag}__SELL` (bool on downtrend flip)
* `{tag}__ST_BUY_LINE` (ST values only during uptrend; for colored line plot)
* `{tag}__ST_SELL_LINE` (ST values only during downtrend)

## Trend Channels (`name="trend_channels"`) — Overlay on price (straight pivot-based lines)

This is the “straight line channel” version using the last 2 pivot highs/lows and extending forward.

### Configs

* `pivot_lookback` (int, default `8`)
  Pivot detection strength.

* `min_pivot_separation_bars` (int, default `5`)
  Prevents pivots from being too close together (reduces noise).

* `show_mid` (bool, default `True`)
  Draw midline (average of upper/lower).

* `show_breakout_markers` (bool, default `True`)
  Mark breakouts above upper / below lower.

* `breakout_marker_size` (int, default `14`)
  Size of breakout markers.

### Output DF columns

* `{tag}__UP_LINE`
* `{tag}__LO_LINE`
* `{tag}__MID_LINE`
* `{tag}__UP_SLOPE` (slope of current upper line)
* `{tag}__LO_SLOPE` (slope of current lower line)
* `{tag}__BREAK_UP` (bool: close > upper line)
* `{tag}__BREAK_DN` (bool: close < lower line)

## Support/Resistance (`name="support_resistance"`) — Overlay on price

This one finds pivot highs/lows, clusters them into horizontal levels, and draws them.

### Configs

* `pivot_lookback` (int, default `10`)
  Pivot strength.

* `tolerance_pct` (float, default `0.003`)
  Clustering tolerance.
  Example: `0.003` = 0.3%.
  Higher = fewer, broader levels; lower = more granular.

* `max_levels` (int, default `8`)
  Maximum number of S/R levels to plot (highest “touch count” clusters).

* `min_touches` (int, default `2`)
  Minimum pivot hits required for a level to qualify.

* `show_break_markers` (bool, default `True`)
  Marks when resistance/support is broken.

* `marker_size` (int, default `14`)
  Break marker size.

### Output DF columns

* `{tag}__SUPPORT` (nearest support ≤ close)
* `{tag}__RESISTANCE` (nearest resistance ≥ close)
* `{tag}__DIST_SUP_PCT` ((close/support - 1) * 100)
* `{tag}__DIST_RES_PCT` ((resistance/close - 1) * 100)
* `{tag}__BREAK_SUP` (bool: broke below support)
* `{tag}__BREAK_RES` (bool: broke above resistance)

## Practical mapping for your strategy rules

Because of tagging, your strategy rules become very explicit and composable:

* RSI14 value: `rsi14__RSI`
* RSI9 value: `rsi9__RSI`
* Bull divergence present: `rsi14__BULL_DIV == True`
* Divergence started in oversold: `rsi14__BULL_START_RSI <= 20`
* Close below BB upper: `close < bb20__UP`
* Supertrend buy flip: `st10_3__BUY == True`
* Price broke channel upper: `tc__BREAK_UP == True`
* Close near resistance: `sr__DIST_RES_PCT < 0.2` (example threshold)
